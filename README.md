## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Add your NewsAPI key in `src/utils/api.js`

3. Start the development server:
   ```bash
   npx expo start
   ```

## Screens

- **City Selector**: Choose your city
- **News Feed**: Browse city news
- **News WebView**: Read full articles
- **Bookmarks**: View saved articles
- **Emergency Alerts**: View local alerts

## API

Uses NewsAPI.org for news data. Get your API key at https://newsapi.org
